# Sigil

An simple functional programming language, written in Kotlin.

[Demo](https://www.youtube.com/watch?v=h87cfyMY-ZA)

## Examples
```ocaml
let factorial n =
	if == 0 n
		1
	* n factorial - n 1

let main =
    print factorial 7
```
